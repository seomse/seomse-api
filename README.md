# seomse-api

# 개발환경
- open jdk 1.8

# 구성
### api 
- java class 구현에 따른 기능 제공 

### 역 제어 api(reverse control api)
- 클라이언트가 서버가 되어 나를 제어할 수 있는 api 기능 제공

### push, receive
- 대량 메시지 전송을 위한 방식, 데이터 전송 단위 크기를 정할 수 있고 메시지 종료 이벤트를 받을 수 있음.
